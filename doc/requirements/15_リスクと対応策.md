# 15. リスクと対応策

| リスク | 影響度 | 発生確率 | 対応策 |
|---|---|---|---|
| OpenAI API障害 | 高 | 低 | フォールバックLLMの検討、エラーメッセージの改善 |
| OpenAI APIコスト増大 | 中 | 中 | 利用量モニタリング、レート制限の実装 |
| Supabase障害 | 高 | 低 | ヘルスチェック監視、状況通知体制の構築 |
| 不適切なAI応答 | 中 | 中 | システムプロンプトの改善、コンテンツフィルタリング |
| β版での予期せぬ負荷 | 低 | 低 | 20ユーザー想定のため低リスク、監視体制の構築 |
| セキュリティインシデント | 高 | 低 | RLS + 監査ログ + CSP による多層防御 |
