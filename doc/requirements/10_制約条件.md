# 10. 制約条件

### 技術的制約

| 制約 | 詳細 |
|---|---|
| 画像ファイルサイズ | 1ファイルあたり最大 5MB（5,242,880 bytes） |
| 画像MIME制限 | `image/jpeg`, `image/png`, `image/webp` のみ |
| 添付枚数制限 | 1メッセージあたり最大3枚 |
| 会話一覧ページサイズ | 1〜50件/ページ |
| 管理者検索ページサイズ | 1〜50件/ページ |
| CSVインポート | 1リクエスト500行以内を推奨 |
| 会話タイトル | ユーザーメッセージの先頭50文字 |
| 署名URL有効期限 | アップロード用: 60秒、表示用: 10分（推定） |
| 月次レポート最大トークン | REPORT_MAX_TOKENS_OUT（デフォルト: 2,000） |

### 依存ライブラリの主要バージョン制約

| ライブラリ | バージョン |
|---|---|
| Next.js | 14.2.35 |
| React | 18.3.1 |
| TypeScript | 5.4.5 |
| Supabase JS | 2.45.0 |
| Vercel AI SDK | 6.0.33 |
| Zod | 4.3.5 |

### ビジネス制約

- β版として約20名の生徒を想定
- OpenAI GPT-4o-mini を使用（コスト最適化のため）
- 塾の生徒のみ利用可能（Allowlistによる制限）

### 法的制約

- 特定のライセンス情報はpackage.jsonに記載なし（オープンソースライセンスの確認が推奨される）
