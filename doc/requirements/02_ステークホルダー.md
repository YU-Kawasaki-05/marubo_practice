# 2. ステークホルダー

| ステークホルダー | 役割 | 期待事項 |
|---|---|---|
| 生徒（中高生） | エンドユーザー（`student`ロール） | いつでも学習質問ができ、分かりやすい回答を即座に得られること |
| 講師・管理スタッフ | 管理者（`staff`ロール） | 生徒の利用状況を把握し、許可リストや権限を管理できること |
| システム管理者 | スーパー管理者（`GRANT_ALLOWED_EMAILS`） | スタッフ権限の付与・剥奪を行い、システム全体を管理できること |
| 塾運営者 | ビジネスオーナー | 生徒の学習支援品質向上とコスト効率の改善 |
