# 11. 前提条件

### 技術的前提条件

#### 必要な環境変数

| 変数名 | 必須 | 説明 |
|---|---|---|
| `NEXT_PUBLIC_SUPABASE_URL` | 必須 | Supabase プロジェクトURL |
| `NEXT_PUBLIC_SUPABASE_ANON_KEY` | 必須 | Supabase 匿名キー |
| `SUPABASE_SERVICE_ROLE_KEY` | 必須 | Supabase サービスロールキー（サーバー専用） |
| `OPENAI_API_KEY` | 必須 | OpenAI APIキー |
| `ADMIN_EMAILS` | 必須 | 管理者メールアドレス（カンマ区切り） |
| `GRANT_ALLOWED_EMAILS` | 必須 | 権限管理許可メール（セミコロン区切り） |
| `NEXT_PUBLIC_SUPPORT_EMAIL` | 任意 | サポート連絡先メール |
| `NEXT_PUBLIC_SUPPORT_FORM_URL` | 任意 | サポートフォームURL |
| `REPORT_LLM_MODEL` | 任意 | レポート生成用LLMモデル |
| `REPORT_LLM_API_KEY` | 任意 | レポート用APIキー（未指定時OPENAI_API_KEY） |
| `REPORT_MAX_TOKENS_OUT` | 任意 | レポート最大出力トークン（デフォルト: 2000） |
| `RESEND_API_KEY` | 任意 | Resend メールサービスキー |
| `MAIL_FROM` | 任意 | メール送信元アドレス |

#### 外部サービスのアカウント

| サービス | 用途 |
|---|---|
| Supabase | 認証、PostgreSQLデータベース、ストレージ |
| OpenAI | AI チャット応答 |
| Vercel | ホスティング・デプロイ |
| Resend | メール送信（将来） |
| GitHub | ソースコード管理・CI |

### 運用上の前提条件

- Node.js LTS + pnpm 9 が利用可能であること
- Supabase プロジェクトが作成済みで、マイグレーションが適用済みであること
- Supabase Storage に `attachments` バケット（private）が作成済みであること
- OpenAI API の利用枠が確保されていること
