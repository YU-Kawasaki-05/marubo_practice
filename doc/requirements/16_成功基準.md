# 16. 成功基準

README に記載されたβ版受入基準（`docs/acceptance.md` 参照）に基づく：

1. 生徒がテキスト・画像でAIに質問でき、ストリーミング応答を受けられること
2. 会話履歴が永続化され、過去の質疑を閲覧できること
3. スタッフが許可リストを管理し、生徒のアクセスを制御できること
4. スタッフが全生徒の会話を検索・閲覧できること
5. 権限管理が適切に機能し、監査ログが記録されること
6. 自動テストが通過し、CI/CD パイプラインが正常に稼働すること
