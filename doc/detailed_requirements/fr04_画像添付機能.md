# FR-04: 画像添付機能

## 4.1 機能概要

チャットメッセージに画像を添付する機能。3ステップ（署名URL発行 → Storageアップロード → チャット送信時に参照）で実装されている。

## 4.2 詳細仕様

### FR-04-4: 署名URL発行

- **エンドポイント**: `POST /api/attachments/sign`
- **処理フロー**:
  1. Bearer トークンで認証
  2. リクエストボディから `mimeType`, `size` を取得
  3. MIME タイプ検証: `image/jpeg`, `image/png`, `image/webp` のみ
  4. ファイルサイズ検証: ≤ 5,242,880 bytes (5MB)
  5. Storage パス生成: `{user_id}/{uuid}.{ext}`
  6. Supabase Storage の `createSignedUploadUrl()` で署名URL発行
  7. `{ signedUrl, storagePath, token }` を返却

### FR-04-5: Storageアップロード

- **クライアント側処理**: 署名URLに対してPUTリクエストで直接アップロード
- **ストレージバケット**: `attachments`（private）
- **パス規約**: `{user_id}/{uuid}.{ext}`
  - `user_id`: Supabase Auth UID
  - `uuid`: `crypto.randomUUID()` で生成
  - `ext`: MIMEタイプから拡張子マッピング

### FR-04-6: 署名付き表示URL

- **生成タイミング**: メッセージ表示時（MessageBubble コンポーネント）
- **有効期限**: 10分（推定）
- **処理**: Supabase Storage の `createSignedUrl()` で生成

### FR-04-8: バリデーション定数

```typescript
// attachmentValidation.ts
const ALLOWED_MIME_TYPES = ['image/jpeg', 'image/png', 'image/webp'] as const
const MAX_FILE_SIZE_BYTES = 5_242_880  // 5MB
const MAX_ATTACHMENTS_PER_MESSAGE = 3

// MIME → 拡張子マッピング
function extFromMimeType(mimeType: string): string {
  switch (mimeType) {
    case 'image/jpeg': return 'jpg'
    case 'image/png':  return 'png'
    case 'image/webp': return 'webp'
    default: throw new AppError(400, 'UNSUPPORTED_MIME_TYPE', ...)
  }
}
```

## 4.3 データ構造

```typescript
// 署名URLリクエスト
type SignRequestBody = {
  filename?: string
  mimeType?: string
  size?: number
}

// 署名URLレスポンス
type SignResponse = {
  requestId: string
  data: {
    signedUrl: string     // アップロード先URL
    storagePath: string   // Storage内パス
    token: string         // アップロードトークン
  }
}

// 添付テーブル行
type AttachmentRow = {
  id: string
  message_id: string
  user_id: string
  storage_path: string       // {user_id}/{uuid}.{ext}
  mime_type: string | null
  size_bytes: number | null
  created_at: string
}
```

## 4.4 エラーハンドリング

| エラーコード | HTTPステータス | 条件 |
|---|---|---|
| UNAUTHORIZED | 401 | 認証失敗 |
| MISSING_MIME_TYPE | 400 | mimeType未指定 |
| MISSING_SIZE | 400 | size未指定 |
| UNSUPPORTED_MIME_TYPE | 400 | 非対応MIME |
| FILE_TOO_LARGE | 400 | 5MB超過 |
| SIGN_URL_FAILED | 500 | 署名URL発行失敗 |
