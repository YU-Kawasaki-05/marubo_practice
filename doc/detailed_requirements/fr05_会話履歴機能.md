# FR-05: 会話履歴機能

## 5.1 機能概要

チャット会話を自動的にデータベースに保存し、サイドバーから過去の会話を閲覧できる。

## 5.2 詳細仕様

### FR-05-1: 会話自動保存

- **タイミング**: AI応答のストリーミング完了時（`onFinish` コールバック）
- **保存手順**:
  1. `conversations` テーブルに新規会話を INSERT
  2. `messages` テーブルにユーザーメッセージ + AI応答を INSERT
  3. 添付がある場合、`attachments` テーブルに INSERT
- **特記**: 保存失敗時もストリーミング応答は正常に返却される（ログ出力のみ）

### FR-05-2: 会話一覧（サイドバー）

- **エンドポイント**: `GET /api/conversations`
- **ページネーション**: カーソルベース
  - カーソル形式: `{created_at}_{id}`
  - デフォルト: 20件/ページ、最大50件
- **ソート**: `created_at DESC`, `id DESC`
- **表示内容**: タイトル、相対日時
- **UI**: サイドバーコンポーネント（ConversationSidebar）

### FR-05-3: 会話詳細閲覧

- **エンドポイント**: `GET /api/conversations/[id]`
- **アクセス制御**: RLS により自身の会話のみ取得可能
- **取得内容**:
  - 会話ヘッダ（id, title, created_at）
  - メッセージ一覧（role, content, created_at）— 昇順
  - 各メッセージの添付画像（storagePath, mimeType, sizeBytes）

### FR-05-4: 相対日時表示

```typescript
// ConversationSidebar.tsx 内のフォーマットロジック
function formatDate(dateStr: string): string {
  const now = new Date()
  const date = new Date(dateStr)
  const diffDays = Math.floor((now.getTime() - date.getTime()) / 86400000)

  if (diffDays === 0) {
    return `今日 ${hours}:${minutes}`
  } else if (diffDays <= 7) {
    return `${diffDays}日前`
  } else {
    return `${year}/${month}/${day}`
  }
}
```

## 5.3 データ構造

```typescript
// 会話サマリー
type ConversationSummary = {
  id: string
  title: string
  created_at: string
}

// 会話一覧レスポンス
type ConversationsListResponse = {
  data: ConversationSummary[]
  nextCursor: string | null
}

// 会話詳細レスポンス
type ConversationDetailResponse = {
  data: {
    id: string
    title: string
    createdAt: string
    messages: Array<{
      id: string
      role: 'user' | 'assistant'
      content: string
      createdAt: string
      attachments: Array<{
        id: string
        storagePath: string
        mimeType: string | null
        sizeBytes: number | null
      }>
    }>
  }
}
```
