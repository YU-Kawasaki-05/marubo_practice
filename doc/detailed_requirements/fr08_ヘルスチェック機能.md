# FR-08: ヘルスチェック機能

## 8.1 機能概要

システムの稼働状態を確認するための軽量エンドポイント。

## 8.2 詳細仕様

- **エンドポイント**: `GET /api/health`
- **Runtime**: Edge Runtime（最小レイテンシ）
- **認証**: 不要
- **レスポンス**: `200 OK`
  ```json
  {
    "status": "ok",
    "timestamp": "2026-03-01T12:00:00.000Z"
  }
  ```

## 8.3 データ構造

```typescript
type HealthResponse = {
  status: 'ok'
  timestamp: string  // ISO 8601
}
```
