# 1. 機能概要

marubo_ai は塾向けAIチャットボットシステム（β版）であり、以下の機能領域で構成される。

| 機能領域 | 概要 | 対象ユーザー |
|---|---|---|
| ユーザー認証 | メール/パスワードによるログイン・サインアップ | 全ユーザー |
| アクセス制御 | 許可リスト（Allowlist）による利用制御 | 全ユーザー（管理はstaff） |
| AIチャット | OpenAI GPT-4o-miniを利用した質疑応答 | student |
| 画像添付 | チャットへの画像アップロード・表示 | student |
| 会話履歴 | 会話の永続化と閲覧 | student |
| 管理：許可リスト | 許可メールの管理・CSV一括インポート | staff |
| 管理：会話検索 | 全生徒の会話検索・閲覧 | staff |
| 管理：権限管理 | スタッフ権限の付与・剥奪 | staff + GRANT_ALLOWED |
| システム管理 | ヘルスチェック・ユーザー同期 | system |
