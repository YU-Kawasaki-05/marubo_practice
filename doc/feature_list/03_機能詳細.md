# 3. 機能詳細

### FR-01: ユーザー認証

| 項目 | 内容 |
|---|---|
| 機能ID | FR-01 |
| 機能名 | ユーザー認証 |
| 説明 | Supabase Authを利用したメール/パスワード認証 |
| 優先度 | 必須（実装済み） |
| 入力 | メールアドレス、パスワード |
| 出力 | 認証トークン（JWT）、ユーザー情報 |
| 関連ファイル | `app/login/page.tsx`, `app/api/sync-user/route.ts`, `src/shared/lib/supabaseClient.ts` |

| サブ機能ID | 機能名 | 説明 | 入力 | 出力 |
|---|---|---|---|---|
| FR-01-1 | サインアップ | メール/パスワードで新規アカウント作成 | email, password | Supabase AuthUser |
| FR-01-2 | ログイン | メール/パスワードでログイン | email, password | Supabase Session (JWT) |
| FR-01-3 | JWTセッション管理 | Bearer トークンによるAPI認証 | Authorization ヘッダ | ユーザーID |
| FR-01-5 | ユーザー同期 | Auth→app_user テーブルの同期、Allowlistチェック | Bearer トークン | appUserId, role, status |

### FR-02: 許可リスト管理

| 項目 | 内容 |
|---|---|
| 機能ID | FR-02 |
| 機能名 | 許可リスト（Allowlist）管理 |
| 説明 | システム利用を許可するメールアドレスの管理 |
| 優先度 | 必須（実装済み） |
| 入力 | メールアドレス、ステータス、ラベル、備考、CSVファイル |
| 出力 | 許可メール一覧、操作結果、監査ログ |
| 関連ファイル | `app/admin/allowlist/page.tsx`, `app/api/admin/allowlist/`, `src/shared/lib/allowlist.ts` |

| サブ機能ID | 機能名 | 説明 | 入力 | 出力 |
|---|---|---|---|---|
| FR-02-1 | 一覧表示 | フィルタ付き許可メール一覧 | status?, search? | AllowedEmailRow[] |
| FR-02-2 | 個別追加 | 新規メール追加 | email, status?, label?, notes? | 作成レコード + 監査ログ |
| FR-02-3 | ステータス更新 | 既存メールの状態変更 | email(path), status?, label?, notes? | 更新レコード + 監査ログ |
| FR-02-4 | CSV一括インポート | CSV形式での一括登録 | csv (text), mode? | inserted/updated件数 |
| FR-02-5 | メール正規化 | 小文字化・トリム | email (raw) | email (normalized) |
| FR-02-6 | 監査ログ | 全操作をaudit_allowlistに記録 | 操作情報 | 監査レコード |
| FR-02-7 | AllowlistGuard | ページアクセス制御コンポーネント | ユーザーセッション | 許可/ブロック表示 |
| FR-02-8 | StatusBanner | ステータス通知バナー | allowlist status | 通知UI |

### FR-03: AIチャット

| 項目 | 内容 |
|---|---|
| 機能ID | FR-03 |
| 機能名 | AIチャット |
| 説明 | OpenAI GPT-4o-miniによるストリーミングチャット |
| 優先度 | 必須（実装済み） |
| 入力 | テキストメッセージ、（任意）添付画像メタデータ |
| 出力 | AI応答（ストリーミング）、会話ID |
| 関連ファイル | `app/api/chat/route.ts`, `src/features/chat/components/ChatInterface.tsx`, `src/shared/utils/ai-message-converter.ts` |

| サブ機能ID | 機能名 | 説明 |
|---|---|---|
| FR-03-1 | テキスト送受信 | ユーザーメッセージ送信→AI応答のストリーミング受信 |
| FR-03-2 | システムプロンプト | 「親切な塾の先生」ロール設定、LaTeX指示 |
| FR-03-3 | LaTeX対応 | 数式を`$...$`/`$$...$$`形式で返却 |
| FR-03-4 | Markdown描画 | GFM + KaTeX + sanitizeでレンダリング |
| FR-03-5 | タイトル自動生成 | ユーザーメッセージ先頭50文字をタイトル化 |
| FR-03-6 | ローディング表示 | ストリーミング中のインジケータ |
| FR-03-7 | 会話管理 | 新規作成・既存会話の選択切替 |

### FR-04: 画像添付

| 項目 | 内容 |
|---|---|
| 機能ID | FR-04 |
| 機能名 | 画像添付 |
| 説明 | チャットへの画像添付（署名URL方式） |
| 優先度 | 必須（実装済み） |
| 入力 | 画像ファイル（JPEG/PNG/WebP、≤5MB） |
| 出力 | 署名URL、Storageパス、添付メタデータ |
| 関連ファイル | `app/api/attachments/sign/route.ts`, `src/features/chat/hooks/useImageAttachments.ts`, `src/shared/lib/attachmentValidation.ts` |

| サブ機能ID | 機能名 | 制約 |
|---|---|---|
| FR-04-1 | ファイル選択 | ファイルピッカーダイアログ |
| FR-04-2 | ドラッグ＆ドロップ | チャットエリアへのD&D |
| FR-04-3 | プレビュー表示 | サムネイル＋削除ボタン |
| FR-04-4 | 署名URL発行 | MIMEチェック + サイズチェック後に発行 |
| FR-04-5 | Storageアップロード | パス: `{user_id}/{uuid}.{ext}` |
| FR-04-6 | 表示URL生成 | 10分有効の署名付き表示URL |
| FR-04-7 | ライトボックス | クリックで拡大表示モーダル |
| FR-04-8 | バリデーション | JPEG/PNG/WebP, ≤5MB, ≤3枚/msg |
